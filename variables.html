Variables in JavaScript

JavaScript uses var, let, const to store values.

Example 1: var, let, const

var x = 10;
let y = 20;
const z = 30;

document.write(x + " " + y + " " + z);

Example 2: Reassign

let name = "Poornima";
name = "Student";
document.write(name);

Example 3: Scope

{
 let a = 5;
}
document.write(a); // error (block scope)
